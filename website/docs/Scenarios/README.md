# SMB customer scenarios

Use these starter scenarios to build future POC-in-a-box modules.

## Scenario backlog

| Scenario | Customer problem | Primary Microsoft capability |
| --- | --- | --- |
| Protect sensitive customer files | Customer records are shared by email and stored in SharePoint | Sensitivity labels and DLP |
| Reduce accidental external sharing | Users need to collaborate externally without oversharing | DLP and sharing governance |
| Encrypt confidential email | Sensitive information must be sent to external recipients | Microsoft Purview Message Encryption |
| Prepare for compliance review | Customer needs a practical readiness path | Compliance Manager and Purview recommendations |

Start with the MIP POC plan, then expand into Defender and broader Purview scenarios.
