# Microsoft Defender Suite for Business Premium

Use this page for the Microsoft Defender add-on motion for SMB customers already on, or moving to, Microsoft 365 Business Premium.

## Partner positioning

Microsoft documentation describes Defender Suite for Business Premium as an add-on that extends Business Premium security capabilities with:

- Microsoft Entra ID P2 for advanced identity security and governance.
- Microsoft Defender for Identity.
- Microsoft Defender for Endpoint Plan 2.
- Microsoft Defender for Office 365 Plan 2.
- Microsoft Defender for Cloud Apps.

## Suggested partner asset angle

Position this as the "security depth" add-on for customers who have outgrown baseline SMB security and need stronger identity, endpoint, email, SaaS, and threat investigation capabilities.
