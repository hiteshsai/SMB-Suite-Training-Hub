# Microsoft Purview Information Protection features

Microsoft Purview Information Protection (MIP) is the first Purview training module for this SMB hub.

## Core Business Premium capabilities

| Capability | Partner-friendly explanation | Example SMB outcome |
| --- | --- | --- |
| Sensitivity labels | Classify and protect email, files, and Teams meeting content | Users can mark files as confidential and apply protection consistently |
| Data Loss Prevention | Detect and help prevent risky sharing of sensitive data | Reduce accidental sharing of customer, payment, or employee data |
| Message Encryption | Protect email messages and attachments | Send confidential information externally with encryption controls |
| Compliance Manager | Highlight recommended improvement actions | Give customers a starting point for compliance readiness conversations |

## Recommended demo storyline

1. Show a sensitive file without labels.
2. Apply a sensitivity label in Microsoft 365 Apps.
3. Show protected sharing behavior.
4. Trigger or explain a DLP policy.
5. Close with next steps: label taxonomy, DLP policy design, user training, and partner-managed deployment.
